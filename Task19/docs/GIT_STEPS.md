# Git Operations Demonstrated

Use these commands from the `FullStack-Tasks` repository root to manage `Task19`.

```bash
git status
git add Task19
git commit -m "Add Task19 sample Git project"
git log --oneline
git remote -v
git push origin main
```

Additional useful commands:

```bash
git init
git add .
git commit -m "Initial commit"
git log --oneline --graph --decorate --all
```
