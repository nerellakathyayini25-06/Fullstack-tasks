# Task 19 Output

## Objective

Create a Git repository for a sample project and demonstrate basic Git operations such as:

- initializing a repository
- staging files
- committing changes
- maintaining version history
- pushing to GitHub
- managing remote repositories

## Sample Project Used

This task includes a sample Python project:

- `app.py`
- `.gitignore`
- `README.md`

## Basic Git Workflow

### Initialize repository

```bash
git init
```

### Check repository status

```bash
git status
```

### Stage files

```bash
git add Task19
```

### Commit changes

```bash
git commit -m "Add Task19 sample Git project"
```

### View version history

```bash
git log --oneline
```

## Remote Repository Management

### Check current remote

```bash
git remote -v
```

### Push to GitHub

```bash
git push origin main
```

## Expected Submission Flow

Because `Task19` is inside the existing `FullStack-Tasks` GitHub repository, the task can be uploaded by:
